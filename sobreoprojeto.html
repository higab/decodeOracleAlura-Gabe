Decodificador Sobre o projeto

Um pouco sobre este projeto e o caminho até aqui:

Olá, bem vindo ao meu projeto de codificação e desconficação de textos. Agradeço a toda a jornada no projeto Oracle Next Education - T6 da parceria Oracle + Alura pois graçaos ao mesmo pude chegar a este momento atual e poder criar meus projetos e portfolios.

Este projeto visa criar um codificador que encritografa/descriptografa o texto inserido no campo a esquerda e exibe o resultado no campo a direita com a possibilidade de copiar o resultado para a área de transferência do dispositivo.
